# fsharp-study-series

## Prerequsits

### In Windows

```pwsh
winget install dotnet-runtime-9
winget install dotnet-sdk-9
winget upgrade

```

## Create a project

```pwsh
dotnet new console -lang F# -o MyFSharpApp
```

## How to run the project

```pwsh
cd MyFSharpApp
dotnet run
```
